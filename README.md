# Logo Quiz

A fun and interactive logo guessing game built with pure HTML, CSS, and JavaScript.

## How to Play

1. Enter your name
2. You have 10 seconds to guess each logo
3. Type your answer and press Submit or Enter
4. See your results at the end!

## Features

- 14 brand logos to guess
- 10-second timer per logo
- Personalized results with your name
- Score tracking and detailed breakdown
- No dependencies - runs entirely in the browser

## Play Now

Visit: [https://revaristo-au-clgx.github.io/logo-quiz/](https://revaristo-au-clgx.github.io/logo-quiz/)
